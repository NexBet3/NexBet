# NexBet - Video Call Platform

A video call website where users must watch ads to make video calls. Users can make 3 calls after watching an ad, then the cycle repeats.

## Features
- **Gmail-only Authentication**: Sign in exclusively through Google Gmail
- **Ad-Based Call System**: Watch ads to unlock 3 video calls
- **Repeating Cycle**: After 3 calls, watch another ad to make more calls
- **Real-time Video Calling**: Peer-to-peer video and audio communication

## Tech Stack
- **Frontend**: React.js with TypeScript
- **Backend**: Node.js/Express.js
- **Database**: MongoDB
- **Real-time Communication**: WebRTC + Socket.io
- **Authentication**: Google OAuth 2.0
- **Ad Service**: Google AdSense / Custom Ad Network
